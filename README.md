# PawPrint - Agent Ops Dashboard

AI agent operations dashboard. Monitor your OpenClaw agents, cron jobs, costs, and errors in one place.

## Packages

- `packages/web` - Next.js dashboard frontend
- `packages/api` - Backend API (TBD)
- `packages/reporter` - Reporter script (TBD)

## Development

```bash
cd packages/web
npm install
npm run dev
```
